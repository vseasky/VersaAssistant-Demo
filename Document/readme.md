# VersaAssistant

## 主题设置

在 `app` 路径的 `theme` 文件夹下创建 `your-theme.json`

```json
{
  "theme-color": [
    "#ffffff",  
    "#1E90FF",  
    "#00BFFF",  
    "#B0C4DE",  
    "#FF7500",  
    "#E60033",  
    "#40DE5A"   
  ]
}
```

其中颜色数组依次对应
1. 背景色
2. 任务栏颜色
3. 侧边栏颜色
4. 其它颜色
5. 停留颜色
6. 按下颜色
7. 选中颜色

::tip

需要注意的是这里颜色数量必须是7个，不能少于这个数。
